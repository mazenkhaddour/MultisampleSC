# Nextflow pipelines for multisamples single cell data 
The pipeline is designed to process multiple single cell samples through quality control, normalization, integration, clustering, and visualization. 
Resuls will be saved as Jupyter notebooks and h5ad files.
it offers a repositories of reusable notebooks, that can be used as a final repo for github.
